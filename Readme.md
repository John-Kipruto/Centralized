# CENTRALIZED

Centralized is a web app that is used by users to create their online profiles and make changes to them when necessary.

## Loom video link

## App Functionalities

* Create user
* Update user
* Delete user

## Technologies used

* React
* Node.js / Express.js
* MySQL
* Redux
* Redux Toolkit
